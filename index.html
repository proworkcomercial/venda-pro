📋 Conectar Google Sheets

Faça isso uma única vez e o ranking vai funcionar para todo mundo.

  1. Acesse sheets.google.com e crie uma planilha nova (pode dar qualquer nome).
  2. No menu superior, clique em Extensões → Apps Script.
  3. Apague tudo que aparecer no editor e cole o código abaixo:
Código — cole no Apps Script
const SHEET = SpreadsheetApp.getActiveSpreadsheet().getSheets()[0]; function doGet() { const val = SHEET.getRange(1,1).getValue(); const json = val || JSON.stringify({members:[],settings:{}}); return ContentService.createTextOutput(json) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { SHEET.getRange(1,1).setValue(e.postData.contents); const out = ContentService.createTextOutput('ok'); out.setMimeType(ContentService.MimeType.TEXT); return out; }
  1. Clique no ícone de salvar 💾 (ou Ctrl+S). Dê qualquer nome ao projeto.
  2. Clique em Implantar → Nova implantação.
  3. Em "Selecionar tipo" escolha Aplicativo da Web.
  4. Em "Executar como" deixe Eu. Em "Quem tem acesso" escolha Qualquer pessoa. Clique Implantar.
  5. Autorize quando pedir (clique em "Permitir").
